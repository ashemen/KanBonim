# Kan Bonim Website Rebuild Planning

This folder contains the first planning package for rebuilding https://www.kanbonim.co.il as a modern Hebrew RTL website with Next.js, TypeScript, and Tailwind CSS.

No website implementation code has been created yet.

## Documents

1. `01-content-audit.md`
   - Existing Wix site inventory
   - Content strengths and weaknesses
   - SEO and conversion opportunities

2. `02-sitemap-strategy.md`
   - Recommended sitemap
   - Page goals
   - Navigation, homepage, services, media, and contact strategy

3. `03-design-system-wireframes.md`
   - Visual direction
   - Color, typography, layout, components
   - Textual wireframes and mobile behavior

4. `04-technical-implementation-plan.md`
   - Next.js architecture
   - Routing, content model, SEO, RTL, assets, contact options
   - Phased implementation plan

5. `05-next-codex-prompt.md`
   - Suggested next prompt to begin implementation safely
